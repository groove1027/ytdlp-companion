## 3. 손익분기점 분석 (⭐ 가장 중요)

### "쓴 만큼만" vs "항상 켜두기" — 언제 뒤집히나?

**RTX 3090/L4 기준 (RunPod)**

```
서버리스 (Flex):  ₩900/시간
월정액 (Pod):     ₩327,000/월 (24시간 상시)

손익분기점: ₩327,000 ÷ ₩900 = 하루 약 12시간

┌─────────────────────────────────────────────────────────┐
│  일 사용량    서버리스 월비용    월정액      어느 게 이득?  │
├─────────────────────────────────────────────────────────┤
│  30분/일     ₩13,500         ₩327,000    서버리스 압승   │
│  1시간/일    ₩27,000         ₩327,000    서버리스 압승   │
│  3시간/일    ₩81,000         ₩327,000    서버리스 이득   │
│  6시간/일    ₩162,000        ₩327,000    서버리스 이득   │
│  12시간/일   ₩324,000        ₩327,000    ← 손익분기점   │
│  18시간/일   ₩486,000        ₩327,000    월정액 이득     │
│  24시간/일   ₩648,000        ₩327,000    월정액 압승     │
└─────────────────────────────────────────────────────────┘
```

> **하루 12시간 이상 GPU를 돌려야 월정액이 이득!**
> 개인/소규모 서비스는 대부분 하루 1~3시간이면 충분 → 서버리스가 유리

## 4. 실제 사용 시나리오별 비용 시뮬레이션

### 시나리오 A: 개인 크리에이터 (하루 영상 1~2개 제작)

| 작업 | 건수/일 | GPU 시간 | 월 GPU 시간 |
|------|--------|---------|------------|
| TTS 음성 합성 (Kokoro/Qwen3) | 20건 | 각 5초 = 100초 | 50분 |
| 음성 인식 (Whisper) | 5건 | 각 2분 = 10분 | 5시간 |
| 합계 | | **~12분/일** | **~6시간/월** |

| 방식 | 월 비용 |
|------|--------|
| **RunPod 서버리스 (3090)** | 6시간 × ₩900 = **₩5,400** |
| iwinv GPU 월정액 | **₩213,150** |
| 현재 API (Typecast 등) | ~₩25,000~43,000 |

→ **서버리스가 ₩5,400으로 압도적으로 저렴!**

## 6. 최종 비용 시뮬레이션 — 3가지 전략

### 전략 A: 카페24 + API 유지 (변화 최소)

```
카페24 VPS (yt-dlp):          ₩7,000/월
기존 API 유지:                 사용량에 따라
────────────────────────────────────
총합:                          ₩7,000 + API 비용
```

- ✅ 간단, 빠른 시작
- ❌ GPU 확장 불가, API 비용 계속 발생

### 전략 B: iwinv + RunPod 서버리스 (확장성 최고) ⭐ 추천

```
iwinv VPS (yt-dlp + Kokoro CPU):   ₩13,100/월 (2GB)
RunPod 서버리스 (Whisper, Qwen3):   ₩5,400~45,000/월 (사용량)
Evolink API (이미지, 대체 불가):     사용량에 따라
Kie/WaveSpeed (대체 불가):          사용량에 따라
────────────────────────────────────────────
총합:                               ₩18,500 + 사용량

나중에 사용량 폭증 시:
→ iwinv GPU 월정액으로 전환 (₩213,150~)
→ 또는 RunPod Active Worker로 할인 (20~30%)
```

- ✅ 지금은 저렴, 나중에 확장 가능
- ✅ 서버리스로 적을 때 싸고, 많아지면 월정액 전환
- ✅ Typecast TTS 비용 절감
- ⚠️ RunPod는 해외 결제 필요

### 전략 C: iwinv 올인원 (한국 결제 Only)

```
iwinv VPS (yt-dlp):                ₩5,600/월 (1GB)
iwinv VPS (Kokoro CPU TTS):        ₩13,100/월 (2GB) — 추후 필요 시
iwinv GPU (Whisper+Qwen3):         ₩213,150/월 — 추후 필요 시
────────────────────────────────────────────────
지금:                               ₩5,600/월
나중에:                             ₩5,600 + ₩213,150 = ₩218,750/월
```

- ✅ 전부 한국 결제
- ✅ 같은 플랫폼에서 확장
- ❌ GPU 쓰기 시작하면 비용 점프 큼 (서버리스 옵션 없음)

## 7. 성장 단계별 로드맵

```
[1단계] 지금 — yt-dlp만 필요
│  iwinv VPS 1GB (₩5,600/월) or 2GB (₩13,100/월)
│  + yt-dlp 서버 설치
│  월 비용: ₩5,600~13,100
│
[2단계] TTS 비용 절감하고 싶을 때
│  같은 VPS에서 Kokoro TTS (CPU) 추가 실행
│  또는 RunPod 서버리스로 Qwen3 TTS
│  Typecast 구독 해지 → 월 ₩25,000~43,000 절감
│  월 비용: ₩13,100 + ₩5,400 = ₩18,500
│
[3단계] Whisper 자동 자막/대사 추출 추가
│  RunPod 서버리스로 Whisper v3 실행
│  리메이크 프리셋에서 원본 영상 대사 자동 추출
│  월 비용: ₩18,500 + ₩5,000~20,000 = ₩23,500~38,500
│
[4단계] 이미지 생성도 자체 호스팅 (사용량 많을 때)
│  RunPod 서버리스로 Stable Diffusion
│  Evolink Nanobanana 비용 절감
│  월 비용: ₩38,500 + ₩20,000~50,000
│
[5단계] 사용량 폭증 (하루 12시간+ GPU)
│  RunPod 서버리스 → iwinv GPU 월정액 전환
│  또는 RunPod Active Worker (30% 할인)
│  월 비용: ₩213,150~ (고정)
```
