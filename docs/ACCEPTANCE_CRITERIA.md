## 🔴 핵심 워크플로우 (반드시 통과)

### TC-001: 대본 → 장면 분할 → 이미지 생성
```
1. 앱 실행 (npm run dev)
2. 대본 모드(SCRIPT) 선택
3. 한국어 대본 10줄 입력
4. 스타일 "시네마틱" 선택, 16:9, Long-form
5. "분석 시작" 클릭

✅ 통과 조건:
- 장면이 3개 이상 생성
- 각 장면에 scriptText, visualPrompt 존재
- 이미지 생성이 시작됨 (isGeneratingImage → true)
- 비용 대시보드에 금액 표시
- 에러 시 alert 또는 Toast로 메시지 표시 (빈 화면 아님)
```

### TC-002: 영상 배치 생성
```
1. TC-001 완료 후 이미지가 있는 프로젝트에서
2. Grok HQ 버튼 클릭 (6초 모드)

✅ 통과 조건:
- batchProgress가 업데이트됨 (current/total 표시)
- 완료된 씬에 videoUrl 존재
- 실패 시 videoGenerationError에 메시지
- 비용 대시보드 업데이트
```

### TC-003: 프로젝트 저장/불러오기
```
1. 프로젝트 작업 후 2초 대기
2. 브라우저 새로고침
3. 사이드바(☰) 열기 → 프로젝트 클릭

✅ 통과 조건:
- 모든 씬 복원 (scriptText, visualPrompt, imageUrl)
- config 설정 복원 (스타일, 비율, 모델 등)
- 썸네일 복원
- 비용 통계 복원
```

### TC-004: HTML 내보내기/가져오기
```
1. 프로젝트 완성 후 "프로젝트 저장 (HTML)" 클릭
2. 다운로드된 HTML 파일을 브라우저에서 열기
3. 같은 HTML 파일을 앱에서 가져오기

✅ 통과 조건:
- HTML 파일이 독립 실행 가능 (씬/이미지/썸네일 표시)
- HTML에서 "이미지 저장" 버튼 동작
- 가져온 프로젝트의 모든 데이터 복원
```

### TC-005: 캐릭터 모드
```
1. 캐릭터 모드(CHARACTER) 선택
2. 캐릭터 라이브러리에서 캐릭터 선택
3. 스타일 선택
4. 생성 클릭

✅ 통과 조건:
- 캐릭터 이미지 생성됨
- 생성된 캐릭터를 대본 모드에서 사용 가능
```
