안녕하세요! AI 기반 영상 제작 올인원 파이프라인, All-in-One Production에 오신 것을 환영합니다.

이 앱은 대본 작성부터 이미지 생성, 영상 제작, 편집, 업로드까지 영상 제작의 전 과정을 하나의 앱 안에서 해결할 수 있도록 만들어진 AI 영상 제작 도구입니다.

이번 영상에서는 회원가입부터 API 키 발급, 앱 설정, 그리고 모든 기능의 사용법까지 A부터 Z까지 빠짐없이 설명해 드리겠습니다.

여섯째, 쿠팡 쇼핑 자동화 설정법.

## PART 2. 회원가입 & 로그인 (약 3분)

### [장면: 로그인 화면]

앱에 처음 접속하시면, 이렇게 로그인 화면이 나타납니다.

참고로, 이 앱은 "소프트 게이트" 방식을 사용합니다. 로그인하지 않아도 앱의 전체 UI를 둘러볼 수 있습니다. 하지만 AI 생성, 영상 렌더링 같은 실제 작업을 실행하려면 로그인이 필요합니다.

상단에 "로그인"과 "회원가입" 두 개의 탭이 보이실 겁니다.

### 2-1. 회원가입

처음 이용하시는 분은 "회원가입" 탭을 클릭해주세요.

회원가입은 총 세 단계로 진행됩니다.

첫 번째 단계, 기본 정보 입력입니다.

이름. 실명을 입력합니다. 필수 항목이에요.
이메일 주소. 로그인에 사용할 이메일을 입력합니다.
비밀번호. 8자 이상으로 설정해주세요. 비밀번호 확인란에 동일하게 한 번 더 입력합니다.
초대 코드. 구매 시 제공받은 초대 코드를 입력해야 합니다. 초대 코드는 자동으로 대문자로 변환되니 소문자로 입력하셔도 괜찮습니다.

두 번째 단계, SMS 본인 인증입니다.

보안과 초대 코드 공유 방지를 위해 전화번호 인증이 필요합니다.

한국 휴대폰 번호를 입력하면, 6자리 인증번호가 문자로 발송됩니다. 수신한 인증번호를 입력하면 본인 인증이 완료됩니다. 재발송은 60초 후에 가능합니다.

세 번째 단계, 가입 완료. "회원가입" 버튼을 누르면 가입이 완료되고, 바로 앱으로 진입합니다.

### 2-2. 소셜 로그인

이메일 회원가입 외에도, 소셜 로그인을 지원합니다.

Google, 카카오, 네이버 세 가지 소셜 로그인이 가능합니다. 해당 버튼을 클릭하면 각 플랫폼의 인증 페이지로 이동하고, 승인 후 바로 앱에 로그인됩니다.

단, 소셜 로그인은 관리자가 해당 OAuth를 설정한 경우에만 버튼이 표시됩니다.

### 2-3. 로그인

이미 계정이 있으시면 "로그인" 탭에서 이메일과 비밀번호를 입력하세요.

"로그인 상태 유지" 체크박스를 켜면 30일 동안 자동 로그인이 유지됩니다. 공용 컴퓨터에서는 체크를 해제하시는 걸 권장합니다.

### 2-4. 프로필 관리

로그인 후에는 우측 상단의 프로필 아이콘을 클릭하면 프로필 모달이 열립니다.

여기서 이름 변경, 비밀번호 변경, 계정 삭제, 로그아웃 등을 할 수 있습니다.

## PART 3. 외부 API 사이트 가입 & 키 발급 (약 12분)

### [장면: API 설정 개요 화면]

이제 가장 중요한 부분입니다. 이 앱은 여러 AI 서비스의 API를 활용해서 동작합니다. 각 서비스에 가입하고 API 키를 발급받아야 앱의 기능을 사용할 수 있습니다.

중요한 점은, 이 앱은 사용자 본인의 API 키를 사용하는 방식입니다. 서비스 비용은 사용자가 각 API 서비스에서 직접 충전하고 관리합니다.

API는 크게 "필수 API" 세 개와 "선택 API" 네 개로 나뉩니다.

필수 API는 Evolink AI, KIE AI, Cloudinary입니다. 이 세 가지가 없으면 앱의 핵심 기능을 사용할 수 없습니다.

선택 API는 Typecast, YouTube Data API, GhostCut, 쿠팡파트너스입니다. 없어도 기본 기능은 동작하지만, 있으면 훨씬 풍부한 기능을 사용할 수 있습니다.

하나씩 가입 방법을 알려드리겠습니다.

---

### 3-1. Evolink AI (필수)

#### [장면: evolink.ai 웹사이트]

Evolink AI는 이 앱의 핵심 엔진입니다. AI 텍스트 분석, 이미지 생성, 영상 생성을 모두 담당합니다.

구체적으로 말씀드리면, Gemini 3.1 Pro 모델로 대본 분석과 장면 분할을 하고, Nanobanana 2 모델로 2K 고화질 이미지를 생성하고, Veo 3.1으로 1080p 고화질 영상을 만듭니다.

가입 방법입니다.

첫째, 브라우저에서 evolink.ai를 접속합니다.
둘째, 우측 상단의 "Sign Up" 또는 "가입" 버튼을 클릭합니다.
셋째, 이메일과 비밀번호로 계정을 생성합니다.
넷째, 대시보드로 이동합니다.
다섯째, API Keys 메뉴에서 새 키를 생성합니다.
여섯째, 생성된 키를 복사합니다. sk-로 시작하는 긴 문자열입니다.

요금은 선불 충전, 즉 크레딧 방식입니다. 이미지 한 장당 약 0.08달러, 즉 한화 약 115원 정도이고, Veo 3.1 영상 하나당 약 0.17달러, 약 245원 정도입니다.

---

### 3-2. KIE AI (필수)

#### [장면: kie.ai 웹사이트]

KIE AI는 Grok 영상 생성, SUNO 음악 생성, ElevenLabs TTS 음성 합성, 그리고 음성 인식 STT를 담당합니다. Evolink가 안 될 때 이미지 생성 백업 역할도 합니다.

가입 방법입니다.

첫째, kie.ai에 접속합니다.
둘째, 계정을 생성합니다.
셋째, 대시보드의 API Key 메뉴로 이동합니다.
넷째, API 키를 생성하고 복사합니다. 32자리 영문+숫자 조합의 문자열입니다.

요금은 역시 크레딧 방식입니다. Grok 영상 6초 기준 약 0.10달러, 한화 약 145원입니다. ElevenLabs 음성 합성은 글자 1000자당 약 0.03달러로 매우 저렴합니다. SUNO 음악 생성은 1곡당 약 0.06달러, 약 85원입니다.

---

### 3-3. Cloudinary (필수)

#### [장면: cloudinary.com 대시보드]

Cloudinary는 이미지와 영상을 인터넷에 호스팅하는 서비스입니다. 생성된 이미지를 저장하고, 영상 생성 AI에 전달할 때 반드시 필요합니다.

좋은 소식은, Cloudinary는 무료 플랜으로도 충분히 사용할 수 있다는 점입니다. 무료로 25GB 저장 공간과 매달 25GB 대역폭을 제공합니다.

가입 방법입니다.

첫째, cloudinary.com에 접속합니다.
둘째, "Sign Up Free"를 클릭해서 무료 계정을 만듭니다.
셋째, 대시보드에 로그인하면 상단에 "Cloud Name"이 표시됩니다. 이걸 메모해주세요. 영문+숫자 조합의 짧은 이름입니다.

넷째, 중요한 단계입니다. Upload Preset을 만들어야 합니다.
Settings, 즉 설정 메뉴로 들어가서, Upload 탭을 클릭합니다.
"Upload Presets" 섹션에서 "Add upload preset"을 클릭합니다.
Signing Mode를 반드시 "Unsigned"로 설정합니다. 이게 핵심입니다.
프리셋 이름은 원하는 대로 지으시면 됩니다. 예를 들어 "storyboard"라고 지을 수 있습니다.
저장 버튼을 누르면 완료입니다.

정리하면, Cloudinary에서 필요한 건 두 가지입니다. Cloud Name과 Upload Preset 이름. 이 두 가지를 앱에 입력하면 됩니다.

---

### 3-4. Typecast (선택)

#### [장면: typecast.ai 웹사이트]

Typecast는 고품질 AI 음성 합성 서비스입니다. 542개 이상의 캐릭터 음성과 26가지 감정 표현을 지원합니다. 60개 이상의 언어를 지원하며, 한국어 감성 나레이션에 특히 강합니다.

기본 TTS 엔진인 ElevenLabs나 Supertonic 2로도 음성을 만들 수 있지만, 더 자연스럽고 감성적인 나레이션이 필요하다면 Typecast를 추천드립니다.

가입 방법은 간단합니다.

typecast.ai/developers에 접속해서 계정을 만들고, 대시보드에서 API 키를 복사하면 됩니다. 키를 발급받을 때 X-API-KEY 형식으로 제공됩니다.

요금은 음성 모델에 따라 다릅니다. v30 고급 모델은 1000자당 약 0.15달러, v21 기본 모델은 1000자당 약 0.075달러입니다.

---

### 3-5. YouTube Data API v3 (선택)

#### [장면: Google Cloud Console]

YouTube API는 채널 분석 탭에서 사용합니다. 경쟁 채널의 구독자 수, 조회수, 인기 영상 등을 분석할 때 필요합니다.

이 API는 Google Cloud에서 무료로 발급받을 수 있고, 하루 10,000 유닛까지 무료입니다. 일반적인 사용에서는 무료 범위를 넘기기 어렵습니다.

가입 및 발급 방법입니다.

첫째, console.cloud.google.com에 접속합니다. Google 계정으로 로그인합니다.
둘째, 상단의 프로젝트 선택 드롭다운에서 "새 프로젝트"를 만듭니다. 이름은 자유롭게 지으세요.
셋째, 좌측 메뉴에서 "API 및 서비스" → "라이브러리"를 클릭합니다.
넷째, "YouTube Data API v3"을 검색해서 "사용" 버튼을 누릅니다.
다섯째, "API 및 서비스" → "사용자 인증 정보"로 이동합니다.
여섯째, 상단의 "사용자 인증 정보 만들기" → "API 키"를 선택합니다.
일곱째, 생성된 API 키를 복사합니다. AIzaSy로 시작하는 문자열입니다.

---

### 3-6. GhostCut (선택)

#### [장면: GhostCut 웹사이트]

GhostCut은 AI 기반 영상 자막 및 워터마크 자동 제거 서비스입니다. 쇼핑 숏폼 제작이나 편집점 매칭 기능에서 소스 영상의 기존 자막을 깔끔하게 지울 때 사용합니다.

이전에 사용하던 ProPainter(Replicate)를 완전히 대체했으며, 더 빠르고 정확한 자막 제거가 가능합니다.

가입 방법입니다.

GhostCut 공식 사이트에서 계정을 만들고, 개발자 대시보드에서 App Key와 App Secret 두 개의 값을 발급받습니다.

인증 방식이 독특합니다. App Key와 App Secret을 조합해서 이중 MD5 서명을 생성하는 방식인데, 앱이 자동으로 처리하니 두 값만 입력하시면 됩니다.

요금은 영상 처리 시간 1초당 약 0.01달러이며, 최소 0.05달러부터 시작합니다.

### 3-7. 쿠팡파트너스 API (선택)

#### [장면: 쿠팡파트너스 대시보드]

쿠팡파트너스 API는 쇼핑 콘텐츠 탭에서 쿠팡 상품의 어필리에이트 링크를 자동 생성할 때 사용합니다.

쿠팡 상품 링크를 넣으면 AI가 분석해서 바이럴 쇼핑 숏폼 대본을 만들어 주는데, 이때 자동으로 수익화 링크까지 생성해줍니다.

가입 방법입니다.

첫째, partners.coupang.com에 접속해서 파트너스 계정을 만듭니다.
둘째, 개발자 도구 메뉴에서 Access Key와 Secret Key를 발급받습니다.

중요한 점이 있습니다. 쿠팡 API는 브라우저에서 직접 호출할 수 없어서, CORS 프록시가 필요합니다. Cloudflare Workers 같은 서비스에 간단한 프록시를 배포해야 합니다. 프록시 배포용 코드는 앱의 docs 폴더에 포함되어 있습니다.

### [장면: 요금 요약표]

여기서 잠깐 전체 요금을 정리해 드리겠습니다.

이미지 생성은 Nanobanana 2 기준 한 장당 약 115원.
영상 생성은 Veo 3.1 1080p 기준 약 245원, Grok 720p 6초 기준 약 145원.
음성 합성은 ElevenLabs 1000자당 약 45원, Typecast는 110~220원.
AI 음악 생성은 SUNO 1곡당 약 85원.
Cloudinary와 YouTube API는 무료.

10장면짜리 영상 하나를 풀파이프라인으로 제작한다고 하면, 이미지 10장에 약 1,150원, 영상 10개에 약 2,450원, 나레이션에 약 100원, AI 분석에 약 70원, 합치면 대략 3,800~4,500원 정도로 고품질 AI 영상 한 편을 만들 수 있습니다.

## PART 4. 앱에 API 키 등록하기 (약 3분)

### [장면: 앱 우측 상단 → API 설정 버튼 클릭]

자, 이제 발급받은 API 키들을 앱에 등록해 보겠습니다.

앱 우측 상단의 톱니바퀴 아이콘, "API 설정" 버튼을 클릭합니다.

설정 모달이 열리면, 위쪽에 "필수" 섹션과 아래쪽에 "선택" 섹션이 보입니다.

### 4-1. 하나씩 직접 입력하기

각 서비스 이름 옆에 입력 칸이 있습니다.

Evolink AI 칸에 sk-로 시작하는 키를 붙여넣습니다.
KIE API 칸에 32자리 키를 붙여넣습니다.
Cloudinary는 Cloud Name과 Upload Preset, 두 칸에 각각 입력합니다.
Typecast는 API Key 하나를 입력합니다.
YouTube API 칸에 AIzaSy로 시작하는 키를 입력합니다.
GhostCut은 App Key와 App Secret, 두 칸에 각각 입력합니다.

쿠팡파트너스는 별도 섹션에서 Access Key, Secret Key, 프록시 URL 세 가지를 입력합니다. "연결 테스트" 버튼으로 프록시가 정상 동작하는지 확인할 수 있습니다.

### 4-2. 스마트 자동 감지 (일괄 등록)

여기서 아주 편리한 기능을 소개해 드립니다. "스마트 자동 감지" 기능입니다.

입력란 아래에 있는 "일괄 붙여넣기" 영역을 펼치세요.

여기에 API 키들을 아무 형식으로나 한꺼번에 붙여넣을 수 있습니다.

예를 들어 이런 형식이 모두 가능합니다.

첫째, JSON 형식. 중괄호 안에 키-값 쌍으로 입력.
둘째, KEY=VALUE 형식. 한 줄에 하나씩 등호로 구분.
셋째, 그냥 키만 나열. 앱이 패턴을 분석해서 어떤 서비스의 키인지 자동으로 추측합니다.

예를 들어, AIzaSy로 시작하면 YouTube API 키로, sk-로 시작하면 Evolink 키로 자동 인식합니다.

텍스트 파일이나 JSON 파일, ENV 파일을 직접 업로드할 수도 있습니다.

"자동 감지" 버튼을 누르면, 감지된 결과가 미리보기로 표시됩니다. 각 키 옆에 어떤 서비스로 매칭되었는지 드롭다운이 나타나는데, 잘못 매칭된 경우 드롭다운으로 직접 서비스를 변경할 수 있습니다.

확인이 끝나면 "입력란에 적용하기" 버튼을 누르세요.

### 4-3. 저장

모든 키가 입력되면, 하단의 "설정 저장 및 적용" 버튼을 클릭합니다.

저장이 완료되면 페이지가 새로고침되면서 새 설정이 적용됩니다.

참고로, 비밀번호처럼 키 값은 기본적으로 마스킹되어 있습니다. "키 표시하기" 버튼을 누르면 전체 키를 확인할 수 있습니다.

### 5-1. 프로젝트 탭 (약 2분)

#### [장면: 프로젝트 대시보드]

프로젝트 탭은 작업 관리의 시작점입니다.

여기서 새 프로젝트를 만들고, 기존 프로젝트를 열고, 관리할 수 있습니다.

"새 프로젝트 만들기"를 클릭하면, 프로젝트 이름을 입력하는 모달이 나타납니다.

프로젝트를 만들지 않고 다른 탭을 클릭하면, 앱이 자동으로 "임시 프로젝트"를 생성합니다. 세션 당 한 번만 자동 생성되니, 바로 작업을 시작할 수 있습니다.

제작 모드는 세 가지가 있습니다.

첫째, "대본 모드". 대본을 직접 작성하거나 AI가 생성한 대본을 기반으로 영상을 만드는 기본 모드입니다.
둘째, "캐릭터 비틀기 모드". 기존 캐릭터 이미지를 다양한 스타일로 변환하는 특수 모드입니다.
셋째, "소스 임포트 모드". 이미 가지고 있는 이미지와 대본을 업로드해서 바로 스토리보드를 구성하는 모드입니다.

화면비도 선택합니다. 가로 16:9, 세로 9:16, 정사각형 1:1, 클래식 4:3 중에서 선택할 수 있습니다. 유튜브용이면 16:9, 숏폼이면 9:16을 추천합니다.

프로젝트 목록은 리스트 뷰와 그리드 뷰로 전환할 수 있고, 수정일순, 이름순, 진행도순으로 정렬할 수도 있습니다. 카드 크기도 소/중/대로 조절 가능합니다.

각 프로젝트 카드에는 현재 모드, 화면비, 장면 수, 예상 영상 길이 등이 배지로 표시됩니다. 프로젝트 선택 후 일괄 삭제도 가능합니다.

외부 HTML 프로젝트 파일을 불러오는 "📥 불러오기" 기능도 있습니다.

### 5-2. 채널/영상 분석 탭 (약 2분)

#### [장면: 채널 분석 화면, 파란색 액센트]

채널 분석 탭에서는 YouTube 채널과 영상을 AI로 분석할 수 있습니다.

이 탭은 세 개의 서브탭으로 구성됩니다.

첫 번째, "키워드 랩". 유튜브 키워드를 검색하고 트렌드를 분석합니다. 검색량, 경쟁 강도, 기회 점수를 파악할 수 있고, 관련 키워드와 상위 영상 목록까지 제공됩니다.

두 번째, "채널 분석실". 유튜브 채널 URL을 입력하면, 구독자 수, 최근 영상 조회수 추이, 업로드 빈도, 콘텐츠 스타일 등을 종합 분석합니다. 분석 결과에서 추출된 "채널 가이드라인"은 나중에 대본 작성 시 AI가 참고하는 톤앤매너 기준이 됩니다.

세 번째, "영상 분석실". 특정 영상의 구조, 흐름, 인기 요인, 조회수, 좋아요, 댓글 수 등 인게이지먼트 지표를 분석합니다. 썸네일 분석 기능도 포함되어 있습니다.

채널 프리셋 기능도 있습니다. 자주 분석하는 채널을 프리셋으로 저장해두면, 다음에 클릭 한 번으로 불러올 수 있습니다.

이 탭을 사용하려면 YouTube Data API v3 키가 등록되어 있어야 합니다.

### 5-3. 대본 작성 탭 (약 3분)

#### [장면: 대본 작성 화면, 보라색 액센트]

대본 작성 탭은 AI가 영상 대본을 써주는 핵심 기능입니다.

두 가지 모드가 있습니다.

첫 번째, "직접 작성". 좌측 텍스트 에디터에 자유롭게 대본을 입력합니다. AI 도움 없이 직접 쓰고 싶을 때 사용합니다.

두 번째, "AI 생성". 3단계 위저드로 AI가 대본을 만들어 줍니다.

1단계, "본능 선택". 여기서 영상의 기본 방향을 설정합니다. "썰형", "빠르게", "자극적" 같은 사전 저장된 프리셋을 선택하거나, 직접 커스텀 프롬프트를 만들 수 있습니다. 이 "본능" 시스템이 AI 대본의 성격과 톤을 결정합니다.

2단계, "주제 추천". AI가 선택한 본능과 채널 가이드라인을 기반으로 5개의 콘텐츠 주제를 추천해줍니다. 각 주제에는 제목, 핵심 포인트, 예상 조회수 등이 포함됩니다. 마음에 드는 주제를 선택하세요.

3단계, "대본 작성". AI가 선택한 주제로 영상 대본을 생성합니다. 실시간 스트리밍으로 대본이 작성되는 과정을 볼 수 있습니다.

벤치마크 패널도 있습니다. 여러 AI 생성 대본을 나란히 비교하면서 최적의 대본을 고를 수 있습니다.

완성된 대본은 직접 수정할 수 있고, 글자 수와 예상 읽기 시간이 실시간으로 표시됩니다.

웹 검색 옵션을 켜면, AI가 실시간 웹 정보를 참고해서 더 정확하고 시의성 있는 대본을 작성합니다.

### 5-4. 사운드 스튜디오 탭 (약 3분)

#### [장면: 사운드 스튜디오, 푸시아 액센트]

사운드 스튜디오에서는 나레이션 음성과 배경 음악을 만듭니다.

크게 두 영역으로 나뉩니다.

#### 나레이션 영역

대본을 문장 단위로 자동 분할하고, 각 문장마다 TTS 음성을 선택해서 나레이션을 생성합니다.

TTS 엔진은 세 가지를 지원합니다.

첫째, Typecast. 542개 이상의 캐릭터 음성, 26가지 감정 표현, 60개 이상의 언어를 지원합니다. "Smart Emotion" 기능으로 AI가 자동으로 적절한 감정을 설정합니다. 별도 API 키가 필요합니다.

둘째, ElevenLabs. 자연스러운 다국어 음성 합성. 70개 이상의 언어를 지원하고, 대화 모드에서 기쁨, 슬픔, 분노, 속삭임 등 감정 태그를 설정할 수 있습니다. KIE API를 통해 사용합니다.

셋째, Supertonic 2. 한국어 특화 로컬 모델입니다. 빠르고 무료에 가까우며, 한국어 기본 나레이션에 적합합니다.

각 문장의 음성을 미리 들어보고, 마음에 안 들면 다른 목소리로 변경하거나 속도를 조절할 수 있습니다. 멀티 스피커 설정으로 여러 화자가 번갈아 나레이션하는 것도 가능합니다.

긴 텍스트는 자동으로 안전한 길이로 분할되어 처리됩니다.

#### 음악 생성 영역

AI가 배경 음악을 만들어 줍니다. SUNO AI를 사용하며, 여러 모델을 지원합니다. V4, V4.5, V4.5+, V4.5ALL, V5 중에서 선택할 수 있습니다.

50개 이상의 장르, 80개 이상의 분위기 태그, 60개 이상의 악기 옵션, BPM 프리셋을 조합해서 원하는 음악을 만들 수 있습니다.

가사를 직접 입력할 수도 있고, AI에게 자동 생성을 맡길 수도 있습니다.

자동 연장 기능으로 원하는 길이까지 음악을 늘릴 수 있고, 보컬 분리 기능으로 목소리와 반주를 따로 추출할 수도 있습니다.

### 5-5. 이미지/영상 탭 (약 3분)

#### [장면: 이미지/영상 탭, 오렌지색 액센트]

이미지/영상 탭은 파이프라인의 핵심입니다. AI가 대본의 각 장면에 맞는 이미지를 생성하고, 그 이미지를 기반으로 영상을 만듭니다.

두 개의 하위 패널이 있습니다.

#### 스타일 선택 패널

영상의 비주얼 스타일을 고릅니다. 30가지 이상의 프로페셔널 스타일 프리셋이 있습니다. 예를 들면 "시네마틱", "애니메이션", "수채화", "미니멀 일러스트" 등입니다.

캐릭터 레퍼런스 이미지를 등록할 수도 있습니다. 캐릭터 사진을 업로드하면 AI가 모든 장면에서 일관된 캐릭터를 유지합니다.

#### 스토리보드 패널

이게 메인입니다.

AI가 대본을 분석해서 장면별로 분할합니다. 영상 포맷에 따라 분할 리듬이 달라집니다.

"LONG" 포맷은 2문장 단위의 자연스러운 흐름. "SHORT" 포맷은 절/구 단위의 빠른 템포. "NANO" 포맷은 단어/비트 단위의 초고속 플래시 컷. "MANUAL" 포맷은 사용자가 직접 장면을 구분합니다.

각 장면 카드에서 할 수 있는 작업은 다음과 같습니다.

이미지 생성. 버튼 하나로 Nanobanana 2 AI가 장면에 맞는 2K 고화질 이미지를 그립니다. 마음에 안 들면 "재생성" 버튼으로 다시 만들 수 있습니다. 웹 검색 모드를 켜면 실제 사물이나 장소를 참고한 이미지를 만들 수 있습니다.

영상 생성. 생성된 이미지를 기반으로 영상을 만듭니다. 두 가지 엔진을 선택할 수 있습니다. Veo 3.1은 1080p 고화질 6초, Grok은 720p로 6초/10초/15초 중 선택합니다.

배치 작업 기능도 있습니다. 상단의 배치 버튼을 누르면 모든 장면의 이미지나 영상을 한꺼번에 자동 생성합니다. 실패한 장면만 모아서 재시도하는 기능도 있습니다.

카메라 앵글과 카메라 무빙 설정도 가능합니다. 클로즈업, 와이드, 패닝, 줌인 등을 장면마다 지정할 수 있습니다.

장면 분할, 합치기, 추가, 삭제, 순서 변경도 자유롭게 할 수 있습니다.

### 5-6. 편집실 탭 (약 4분)

#### [장면: 편집실 화면, 앰버 액센트]

편집실에서는 생성된 장면들을 하나의 영상으로 조합하고 편집합니다.

두 개의 서브탭이 있습니다.

#### 타임라인 에디터

장면들을 타임라인에 배치하고 편집하는 메인 에디터입니다.

자막 스타일링. 30가지 이상의 자막 템플릿을 적용할 수 있습니다. 기본 스타일부터 스티커, 네온, 메탈, 글리치 등 다양한 효과가 있습니다. 폰트도 Pretendard, 세리프, 브러시, 고딕 등을 선택할 수 있고, 크기, 색상, 위치를 자유롭게 설정합니다. 페이드인, 바운스, 슬라이드, 줌, 플립 등 자막 애니메이션도 적용 가능합니다.

모션 이펙트. 20가지 이상의 장면 모션을 적용할 수 있습니다. 켄 번즈 효과, 줌인/아웃, 돌리 줌, 회전, 흔들림 등이 있습니다.

트랜지션. 장면과 장면 사이에 20가지 이상의 전환 효과를 넣을 수 있습니다. 크로스페이드, 와이프, 디졸브, 슬라이드 등입니다.

오디오 믹싱. 나레이션, BGM, 효과음을 멀티 트랙으로 믹싱합니다. 장면별 볼륨 조절, 노멀라이즈, 페이드 인/아웃, EQ 조절이 가능합니다.

세이프존 오버레이 기능도 있습니다. YouTube Shorts, Instagram Reels, TikTok 등 플랫폼별 UI 영역을 미리보기에서 확인할 수 있어, 중요한 자막이나 요소가 잘리지 않는지 체크할 수 있습니다.

렌더링은 브라우저의 WebCodecs 기술을 사용합니다. 서버에 영상을 업로드하지 않아도 되고, 워터마크도 붙지 않습니다. MP4, SRT 자막 파일, HTML+ZIP 아카이브로 내보낼 수 있습니다.

#### 편집점 매칭

외부 영상 소스를 활용할 때 쓰이는 3단계 워크플로우입니다.

1단계, 소스 영상 파일과 편집 결정표(EDL) 또는 대본 타임라인을 등록합니다.
2단계, AI가 편집점 타임코드를 자동으로 감지하고, 장면 경계를 정제합니다. 수동으로 미세 조정도 가능합니다.
3단계, 결과를 내보냅니다. 브라우저에서 직접 MP4를 합성하거나, FFmpeg 스크립트, EDL+SRT 파일로 다운로드할 수 있습니다.

여기서 GhostCut AI 자막 제거 기능을 활성화할 수 있습니다. 토글을 켜면, 내보내기 전에 소스 영상의 기존 자막을 AI로 깔끔하게 제거합니다.

### 5-7. 업로드 탭 (약 2분)

#### [장면: 업로드 화면, 초록색 액센트]

업로드 탭에서는 완성된 영상을 각 플랫폼에 업로드합니다.

5개 플랫폼을 지원합니다. YouTube, TikTok, Instagram Reels, Threads, Naver Clip입니다.

6단계 위저드로 구성됩니다.

1단계, 플랫폼 인증. 업로드할 플랫폼을 선택하고 계정을 연결합니다. 각 플랫폼의 OAuth 인증을 통해 안전하게 연동됩니다.

2단계, 영상 선택. 편집실에서 렌더링한 영상을 선택합니다.

3단계, 메타데이터 설정. AI가 자동으로 제목, 설명, 태그를 생성해줍니다. 물론 직접 수정할 수도 있습니다. 플랫폼별로 최적화된 메타데이터를 따로 만들어 줍니다. 쇼핑 콘텐츠의 경우, 상품명 기반의 쇼핑 태그도 자동 생성됩니다.

4단계, 썸네일 선택. 자동 생성된 썸네일 중에서 선택하거나, 인라인 썸네일 스튜디오에서 직접 만들 수도 있습니다.

5단계, 업로드 설정. 공개 범위, 예약 업로드 시간, 카테고리, 재생목록 등을 설정합니다.

6단계, 업로드 실행. 업로드 진행률이 실시간으로 표시됩니다.

### 6-1. 썸네일 스튜디오 (약 1.5분)

텍스트 오버레이 설정에서 폰트, 위치, 크기를 조절하면 AI가 여러 장의 썸네일 시안을 만들어 줍니다.

경쟁 채널의 썸네일을 레퍼런스 이미지로 분석하는 기능도 있습니다. 인기 채널의 썸네일 스타일을 참고해서 비슷한 느낌의 썸네일을 만들 수 있습니다.

### 6-2. 캐릭터 비틀기 (약 1.5분)

#### [장면: 캐릭터 비틀기 탭, 빨간 액센트]

기존 캐릭터 이미지를 다양한 스타일로 변환하는 도구입니다.

두 가지 변환 모드가 있습니다.

"랜덤" 모드는 AI가 자동으로 4가지 다양한 스타일 변형을 만들어 줍니다.
"커스텀" 모드는 사용자가 원하는 스타일을 직접 지정합니다. 예를 들어 수채화, 팝아트, 3D 렌더링, 만화 스타일 등으로 변환할 수 있습니다.

가로, 세로, 정사각형 화면비를 선택할 수 있고, AI가 전신 렌더링까지 완성합니다.

마음에 드는 캐릭터를 "고정"하면, 이후 영상 전반에서 일관된 캐릭터를 유지할 수 있습니다.

별도의 AI 분석 없이 바로 이미지/영상 탭으로 넘어가서 영상 생성을 시작할 수 있어 편리합니다.

### 6-4. PPT 마스터 (약 1.5분)

#### [장면: PPT 마스터 화면]

AI로 프레젠테이션을 자동 생성하는 도구입니다.

4단계로 진행됩니다.

1단계, 텍스트 입력. 발표 주제나 내용을 입력합니다.
2단계, 디자인 스타일 선택. 30가지 이상의 프레젠테이션 템플릿 중에서 선택합니다. 각 템플릿은 배경색과 액센트 색상이 조합되어 있어 미리보기로 확인할 수 있습니다.
3단계, 세부 설정. 슬라이드 수, 언어, 화면비, 색상 등을 조정합니다.
4단계, 생성. AI가 슬라이드 구조를 설계하고, Nanobanana 2로 각 슬라이드에 맞는 비주얼을 생성합니다.

완성된 결과물은 개별 이미지 또는 HTML 프로젝트로 다운로드할 수 있습니다.

### 6-5. 상세 페이지 & 쇼핑 콘텐츠 (약 4분)

#### [장면: 상세 페이지 탭]

이 탭은 이커머스 관련 콘텐츠를 제작하는 종합 도구입니다. 세 개의 서브탭으로 구성됩니다.

#### 서브탭 1: 상세 페이지 빌더

상품 정보를 입력하면 AI가 5~9페이지 분량의 상품 상세 이미지를 자동 생성합니다.

3단계로 진행됩니다.

1단계, 상품 정보 입력. 카테고리, 타겟 고객, 성별, 연령대 등을 설정합니다.
2단계, AI 전략 기획. 상품 유형에 맞는 논리적 섹션 구성을 AI가 자동으로 설계합니다.
3단계, 이미지 생성. Nanobanana 2로 9:16 세로형 상세 이미지를 자동 생성합니다. 5장, 7장, 9장 중 페이지 수를 선택할 수 있습니다.

깔끔한 클린 스타일, 라이프스타일 스타일, 크리에이티브 스타일 등 썸네일 디자인도 선택 가능합니다.

#### 서브탭 2: 썸네일 스튜디오

상세 페이지 전용 썸네일을 만드는 도구입니다.

#### 서브탭 3: 쇼핑 숏폼 자동화

이것이 가장 강력한 기능입니다. 두 가지 소스를 지원합니다.

##### 모드 1: 해외 영상 기반

해외 쇼핑 영상 파일을 업로드하면, AI가 영상을 분석합니다. 음성이 있으면 나레이션도 자동 인식해서 텍스트로 변환합니다.

AI가 분석 결과를 바탕으로 한국어 바이럴 쇼핑 대본을 여러 버전 생성합니다. 각 대본은 "후크 → 디테일 → 로맨스 → 위트+CTA" 4단계 구매 합리화 구조로 작성됩니다.

마음에 드는 대본을 선택한 후, TTS 엔진과 음성을 고르고, 자막 스타일을 설정하고, CTA 문구를 정합니다.

GhostCut AI가 원본 영상의 외국어 자막을 자동으로 제거하고, 한국어 자막과 나레이션을 합성해서 완성된 숏폼 영상을 만들어 줍니다.

##### 모드 2: 쿠팡 링크 기반

쿠팡 상품 URL을 넣으면, CORS 프록시를 통해 상품 페이지를 크롤링합니다. 상품명, 가격, 할인율, 평점, 리뷰 수, 로켓배송 여부, 상품 이미지, 리뷰 내용까지 자동으로 수집합니다.

수집된 데이터를 Gemini 3.1 Pro가 분석해서, "동적 타겟팅 기반 쇼핑형 대본 생성 지침서 v36.0"에 따라 바이럴 쇼핑 대본을 5개 버전 생성합니다.

대본에는 실제 리뷰의 소셜 프루프, 가격 대비 가치 강조, 할인율 임팩트, 로켓배송 언급 등이 자동으로 반영됩니다.

쿠팡파트너스 API가 등록되어 있으면, 어필리에이트 딥링크까지 자동 생성합니다. 링크 복사 버튼으로 바로 CTA에 활용할 수 있습니다.

쿠팡 설정 패널에서는 Access Key, Secret Key, 프록시 URL을 입력하고, "연결 테스트"와 "딥링크 테스트" 버튼으로 설정이 정상인지 확인할 수 있습니다.

### 6-6. 자막/워터마크 제거 (약 2분)

#### [장면: 자막/워터마크 제거 탭]

도구모음의 마지막 기능, 자막과 워터마크 제거입니다. 좌측 사이드바에서 🧹 아이콘의 "자막/워터마크 제거"를 클릭하면 진입합니다.

이 기능은 GhostCut AI 엔진을 사용합니다. 영상에 박혀 있는 외국어 자막, 로고, 워터마크를 AI가 자동으로 감지하고 깔끔하게 제거해 줍니다.

#### 사용 방법

화면은 좌우 2단 레이아웃입니다. 왼쪽은 업로드 영역, 오른쪽은 진행 상황과 결과 영역입니다.

왼쪽 패널에서 영상 파일을 드래그 앤 드롭하거나 클릭해서 선택합니다. MP4, MOV, AVI, WebM 형식을 지원합니다. 파일을 선택하면 파일명, 크기, 예상 비용이 표시됩니다.

"자막/워터마크 제거 시작" 버튼을 클릭하면 처리가 시작됩니다.

#### 처리 과정

오른쪽 패널에 4단계 진행 상황이 표시됩니다.

1단계, 영상 업로드. Cloudinary에 영상을 업로드합니다.
2단계, AI 분석 요청. GhostCut API에 자막 제거 작업을 제출합니다.
3단계, AI 처리 중. GhostCut이 영상을 분석하고 자막과 워터마크를 제거합니다. 영상 길이에 따라 수 분이 소요될 수 있습니다.
4단계, 완료. 처리된 영상을 다운로드할 수 있습니다.

처리가 완료되면 결과 영상이 미리보기로 표시되고, "결과 영상 다운로드" 버튼으로 바로 저장할 수 있습니다. 비용은 영상당 약 0.5달러 수준이며, 실시간 비용 추적에 자동 반영됩니다.

이 기능은 쇼핑 숏폼 자동화에서도 자동으로 호출되지만, 단독으로도 사용할 수 있어서 해외 영상 소스의 자막 제거가 필요할 때 편리합니다.

## PART 7. 실시간 비용 추적 & 팁 (약 2분)

### [장면: 하단 비용 패널]

마지막으로 알아두면 유용한 기능들을 소개합니다.

### 7-1. 실시간 비용 추적

앱 하단에 실시간 비용 표시가 있습니다. 이미지를 생성하거나 영상을 만들 때마다 사용된 비용이 달러와 원화로 표시됩니다. 서비스별로 이미지, 영상, TTS, 음악, AI 분석 등 카테고리가 나뉘어 있습니다. 환율은 자동 업데이트됩니다.

### 7-2. 커뮤니티 미디어 통합

이미지 생성 외에도, Klipy, irasutoya, Google Images, MyInstants, SFX Lab 등에서 밈, 일러스트, 효과음을 검색해서 바로 삽입할 수 있습니다.

### 7-3. 데이터 저장

모든 프로젝트 데이터는 브라우저의 IndexedDB에 자동 저장됩니다. 브라우저를 닫아도 작업 내용이 유지됩니다. 단, 브라우저 데이터를 삭제하면 프로젝트도 삭제되니 주의하세요. 중요한 프로젝트는 HTML 내보내기 기능으로 백업해두시는 것을 권장합니다.

### 7-4. 오류 발생 시

작업 중 오류가 발생하면, 우측 하단에 토스트 알림이 표시됩니다. 대부분의 오류는 API 키가 없거나, 크레딧이 부족하거나, 네트워크 문제일 때 발생합니다.

API 잔액이 부족하다는 메시지가 나오면, 해당 API 서비스 대시보드에서 크레딧을 충전하시면 됩니다.

## PART 8. 피드백 & 오류 신고 (약 2분)

### [장면: 상단 툴바 → "피드백" 버튼 클릭]

마지막으로 중요한 안내를 드리겠습니다.

이 앱은 지속적으로 고도화해 나가고 있는 서비스입니다. 새로운 AI 모델이 나오면 적용하고, 기능도 계속 추가하고, 사용성도 개선해 나갈 예정입니다.

다만 그 과정에서 예상치 못한 오류나 버그가 발생할 수 있습니다. 이럴 때 가장 큰 도움이 되는 것이 바로 여러분의 피드백입니다.

앱 상단 우측에 "피드백" 버튼이 있습니다. 이 버튼을 클릭하면 피드백 모달이 열립니다.

### [장면: 피드백 모달 화면]

피드백은 세 가지 유형으로 보낼 수 있습니다.

첫째, "버그/오류". 앱이 비정상적으로 동작하거나, 에러가 발생한 경우에 선택합니다.
둘째, "제안". 이런 기능이 있으면 좋겠다, 이 부분이 불편하다 등 개선 아이디어를 보낼 때 선택합니다.
셋째, "기타". 질문이나 기타 사항을 전달할 때 사용합니다.

내용을 작성하실 때는 가능한 한 구체적으로 적어주시면 큰 도움이 됩니다.

예를 들어 버그를 신고하실 때는, "어떤 탭에서", "어떤 버튼을 눌렀을 때", "어떤 오류 메시지가 나왔는지"를 함께 적어주시면 원인 파악이 훨씬 빨라집니다.

스크린샷 첨부 기능도 있습니다. 오류 화면을 캡처해서 최대 3장까지 첨부할 수 있습니다. 파일 업로드뿐만 아니라, 스크린샷을 클립보드에 복사한 후 Ctrl+V로 바로 붙여넣기도 가능합니다.

피드백에는 자동으로 사용자 정보가 첨부되어, 빠른 추적과 대응이 가능합니다.

답변받을 이메일 주소를 입력하시면, 처리 결과를 이메일로 회신드리기도 합니다.

보내주신 피드백은 하나하나 확인하고 있으며, 최대한 빠르게 반영하여 오류를 수정하고 기능을 개선하겠습니다. 여러분의 피드백이 이 앱을 더 좋게 만드는 가장 큰 원동력입니다.
