# Typecast 음성 다운로드 옵션 기술 문서

> **소스 페이지:** `https://typecast.ai/text-to-speech/{project_id}`
> **모달 트리거:** 우측 상단 "다운로드" 버튼 클릭 시 표시

---

## 1. 모달 개요

"다운로드" 버튼을 클릭하면 **"음성 다운로드"** 모달 다이얼로그가 표시됩니다. 이 모달에서는 오디오 품질, 파일 형식, 다운로드 범위, 문장 통합 여부를 설정한 뒤 음성 파일을 다운로드할 수 있습니다.

---

## 2. 다운로드 설정 옵션

### 2.1 오디오 품질

드롭다운 선택 방식 (combobox)

| 옵션 | 설명 |
|------|------|
| **높음** | 고품질 오디오 출력 (기본값) |
| **보통** | 일반 품질 오디오 출력 |

- **UI 컴포넌트:** React Select (combobox)
- **기본 선택값:** `높음`

---

### 2.2 파일 형식

드롭다운 선택 방식 (combobox)

| 옵션 | 설명 |
|------|------|
| **mp3** | MPEG Layer III 압축 포맷. 파일 크기가 작아 웹 스트리밍 및 배포에 적합 (기본값) |
| **wav** | PCM 비압축 포맷. 높은 오디오 품질로 전문 프로덕션에 적합 |

- **UI 컴포넌트:** React Select (combobox)
- **기본 선택값:** `mp3`

---

### 2.3 다운로드 범위

라디오 버튼 그룹 방식 (radiogroup)

| 옵션 | 값 (value) | 설명 |
|------|-----------|------|
| **전체** | `all` | 프로젝트 내 모든 문장을 다운로드 (기본값) |
| **선택 문장** | `selected` | 사용자가 선택한 특정 문장만 다운로드 |

- **UI 컴포넌트:** Radio Group
- **파라미터명:** `download_range`
- **기본 선택값:** `all` (전체)

---

### 2.4 문장 통합 여부

라디오 버튼 그룹 방식 (radiogroup)

| 옵션 | 값 (value) | 설명 |
|------|-----------|------|
| **한 파일로 합치기** | `whole` | 모든 문장을 하나의 오디오 파일로 병합하여 다운로드 (기본값) |
| **문장 별로 나누기** | `divided` | 각 문장을 개별 오디오 파일로 분리하여 다운로드 |

- **UI 컴포넌트:** Radio Group
- **파라미터명:** `split_options`
- **기본 선택값:** `whole` (한 파일로 합치기)

---

## 3. 다운로드 실행

모든 옵션 설정 후 하단의 **"다운로드"** 버튼(submit)을 클릭하여 음성 파일 다운로드를 실행합니다.

- **버튼 타입:** `submit` (form 제출)
- **동작:** 설정된 옵션에 따라 음성 파일 생성 및 다운로드 시작

---

## 4. 예상 차감 시간 (Footer 정보)

모달 하단 푸터 영역에 다운로드 시 소비될 예상 시간이 표시됩니다.

| 항목 | 내용 |
|------|------|
| **예상 차감 시간** | `약 NN분 NN초` 형식으로 표시 (프로젝트 텍스트 분량에 따라 변동) |
| **안내 문구 1** | "예상 차감 시간과 실제 차감 결과는 다를 수 있어요." |
| **안내 문구 2** | "본 서비스는 생성형 AI 기반으로 운영되며 AI 음성 합성, AI 토킹 아바타 영상 생성 결과물은 인공지능에 의해 생성된 결과물입니다." |

> **참고:** 예상 차감 시간은 사용자의 구독 플랜에서 차감되는 시간을 의미하며, 실제 다운로드 소요 시간과는 다를 수 있습니다.

## 5. 모달 UI 구조 요약

```
┌─────────────────────────────────────────┐
│  음성 다운로드                      [X] │  ← Header + Close 버튼
├─────────────────────────────────────────┤
│                                         │
│  오디오 품질        [높음        ▾]     │  ← Combobox (높음 / 보통)
│                                         │
│  파일 형식          [mp3         ▾]     │  ← Combobox (mp3 / wav)
│                                         │
│  다운로드 범위                          │
│  (●) 전체    (○) 선택 문장             │  ← Radio Group
│                                         │
│  문장 통합 여부                         │
│  (●) 한 파일로 합치기                   │  ← Radio Group
│  (○) 문장 별로 나누기                   │
│                                         │
│         [ 다운로드 ]                    │  ← Submit 버튼
│                                         │

│  예상 차감 시간                         │
│  └ 약 15분 17초                         │  ← Footer

│    다를 수 있어요.                      │
│  ⓘ 본 서비스는 생성형 AI 기반으로...   │
└─────────────────────────────────────────┘
```

총 가능한 다운로드 설정 조합은 다음과 같습니다:

| 오디오 품질 | 파일 형식 | 다운로드 범위 | 문장 통합 | 가능 여부 |
|------------|----------|-------------|----------|----------|
| 높음 | mp3 | 전체 | 한 파일로 합치기 | ✅ (기본값) |
| 높음 | mp3 | 전체 | 문장 별로 나누기 | ✅ |
| 높음 | mp3 | 선택 문장 | 한 파일로 합치기 | ✅ |
| 높음 | mp3 | 선택 문장 | 문장 별로 나누기 | ✅ |
| 높음 | wav | 전체 | 한 파일로 합치기 | ✅ |
| 높음 | wav | 전체 | 문장 별로 나누기 | ✅ |
| 높음 | wav | 선택 문장 | 한 파일로 합치기 | ✅ |
| 높음 | wav | 선택 문장 | 문장 별로 나누기 | ✅ |
| 보통 | mp3 | 전체 | 한 파일로 합치기 | ✅ |
| 보통 | mp3 | 전체 | 문장 별로 나누기 | ✅ |
| 보통 | mp3 | 선택 문장 | 한 파일로 합치기 | ✅ |
| 보통 | mp3 | 선택 문장 | 문장 별로 나누기 | ✅ |
| 보통 | wav | 전체 | 한 파일로 합치기 | ✅ |
| 보통 | wav | 전체 | 문장 별로 나누기 | ✅ |
| 보통 | wav | 선택 문장 | 한 파일로 합치기 | ✅ |
| 보통 | wav | 선택 문장 | 문장 별로 나누기 | ✅ |

> **총 조합 수:** 2 (품질) × 2 (형식) × 2 (범위) × 2 (통합) = **16가지**
