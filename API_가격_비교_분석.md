## 3. 이미지 생성 — Nanobanana 2 (동일 모델 비교)

**모델: `gemini-3.1-flash-image-preview` (Google이 만든 모델, 브랜드명 "Nanobanana 2")**

Nanobanana 2는 프록시 자체 모델이 아니라 **Google DeepMind의 Gemini 이미지 생성 모델**입니다.
Google Gemini API에서 직접 사용 가능합니다.

| 해상도 | 이 앱 (Evolink 프록시) | Google Gemini API 직접 | 비교 |
|--------|----------------------|----------------------|------|
| 0.5K (512px) | $0.08 | **$0.045** | 공식이 44% 저렴 |
| 1K (1024px) | $0.08 | **$0.067** | 공식이 16% 저렴 |
| **2K (2048px)** | **$0.08** | $0.101 | **이 앱이 21% 저렴** |
| 4K (4096px) | $0.08 | $0.151 | **이 앱이 47% 저렴** |

### 결론: 해상도에 따라 다름

> 이 앱은 **해상도에 관계없이 균일 $0.08**이므로,
> 저해상도(0.5K~1K)에서는 공식이 저렴하고, **고해상도(2K~4K)에서는 이 앱이 훨씬 저렴.**
>
> 이 앱에서 기본 사용하는 2K 기준으로는 **이 앱이 21% 저렴.**

**공식 가격 계산 근거:** Gemini API 이미지 출력 = $60/1M output tokens. 2K 이미지 ≈ 1,680 tokens → $0.101/장

**출처:** [Nano Banana | Gemini API 공식 문서](https://ai.google.dev/gemini-api/docs/nanobanana), [Gemini API Pricing](https://ai.google.dev/gemini-api/docs/pricing)

## 5. TTS — ElevenLabs (동일 모델 비교)

**모델: `elevenlabs/text-to-dialogue-v3` (Turbo 2.5 / Multilingual V2)**

ElevenLabs 공식은 **구독 + 초과분 종량제** 구조. API는 별도 비용 없이 모든 플랜에서 사용 가능하며, 문자 크레딧에서 차감.

| 항목 | 이 앱 (Kie 프록시) | ElevenLabs 공식 (초과분 종량제) |
|------|-------------------|---------------------------|
| **Turbo 2.5** | **$0.03/1K자** | **$0.18~$0.30/1K자** |
| **Multilingual V2** | **$0.06/1K자** | **$0.18~$0.30/1K자** |

**ElevenLabs 공식 초과 요금 (플랜별):**

| 플랜 | 월 가격 | 기본 문자 수 | 초과분 가격 (/1K자) |
|------|---------|------------|------------------|
| Starter | $5 | 30,000 | $0.30 |
| Pro | $99 | 500,000 | $0.24 |
| Scale | $330 | 2,000,000+ | $0.18 |
| Business | $1,320 | 수백만 | $0.12 |

### 절감률: 83~90% (Turbo 기준, Scale 플랜 초과분 대비)

> ElevenLabs 공식에서 가장 저렴한 초과 요금($0.12/1K자, Business)과 비교해도
> 이 앱 Turbo $0.03은 **75% 저렴.**
>
> 일반 사용자가 접근 가능한 Scale($0.18/1K자) 대비로는 **83% 저렴.**
>
> 게다가 공식은 **월 구독료($5~$1,320)를 먼저 내야** 초과 종량제를 쓸 수 있음.
> 이 앱은 **구독 없이 바로 종량제** 사용 가능.

**출처:** [ElevenLabs API Pricing](https://elevenlabs.io/pricing/api), [ElevenLabs Usage Based Billing](https://help.elevenlabs.io/hc/en-us/articles/24351324241937)

## 6. 음악 생성 — Suno (공식 API 없음)

**모델: `suno` (Kie 프록시)**

| 항목 | 이 앱 (Kie 프록시) | Suno 공식 |
|------|-------------------|----------|
| 가격 | **$0.06/곡** | **공식 API 없음** |
| 연장(Extend) | $0.06/회 | - |
| 접근 방식 | REST API | 웹앱만 |

### Suno는 공식 공개 API를 제공하지 않는다

> Suno 구독($10/월 Pro, $30/월 Premier)은 **웹앱 사용권일 뿐 API 접근이 아닙니다.**
> Suno를 **API로 자동화**하려면 Kie 같은 프록시 서비스가 **유일한 경로**입니다.
>
> 따라서 "Suno 구독이 더 저렴하다"는 비교 자체가 성립하지 않습니다.
> 구독은 수동으로 웹에서 하나씩 만드는 것이고, 이 앱은 API로 자동 생성+자동 연장까지 합니다.

**Suno 구독 가격 (참고용, API 아님):**

| 플랜 | 월 가격 | 곡 수 | 곡당 환산 | API 접근 |
|------|---------|------|----------|---------|
| Free | $0 | ~300/월 | 무료 | 없음 |
| Pro | $10 | ~500/월 | $0.02 | **없음** |
| Premier | $30 | ~2,000/월 | $0.015 | **없음** |
| **이 앱** | 종량제 | 무제한 | **$0.06** | **API 자동화** |

**출처:** [Suno Pricing](https://suno.com/pricing), [Suno API 현실 분석 (AIML API)](https://aimlapi.com/blog/the-suno-api-reality)

| 항목 | 이 앱 (10편) | 공식 직접 (10편) | 절감 |
|------|-------------|----------------|------|

## 유의사항

1. 공식 가격은 수시로 변동됩니다. Google은 2025년 하반기부터 AI 서비스 가격 인하 추세.
2. 프록시 API는 공식 API를 중개하므로, 공식 정책 변경 시 가격/가용성에 영향 가능.
3. ElevenLabs 공식은 구독 기본 문자 내에서는 더 저렴할 수 있으나, **초과분 종량제** 기준으로 비교.
4. 이 앱의 핵심 장점: **구독 없이 종량제**로 모든 서비스 통합 사용 가능.
