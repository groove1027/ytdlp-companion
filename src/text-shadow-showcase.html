Implementation Guide
CSS Integration: Copy any effect from advanced-text-shadow-recipes.css and apply to your subtitle elements. Each effect is a self-contained CSS class.
<h2 class="text-shadow-gold-metallic-3d" style="font-size: 56px; font-weight: 700;"> Your Subtitle Text </h2>
Font Recommendations:
  • Font size: 48px-64px (optimal for effect visibility)
  • Font weight: 600-700 (semi-bold to bold)
  • Font family: Sans-serif (Roboto, Helvetica, Arial)
  • Letter spacing: 0-0.05em
  • Line height: 1.2-1.4
Background Optimization:
  • #0a0a0a — Deep black (recommended)
  • #1a1a1a — Dark gray
  • #0d1117 — GitHub dark
  • Dark gradients from #000 to #1a1a1a
Performance Notes:
  • Text-shadow is GPU-accelerated (no performance impact)
  • 6-10 shadows per element is optimal
  • Avoid animating text-shadow with transitions
  • All modern browsers fully supported (Chrome 26+, Firefox 3.1+, Safari 5.1+)
Responsive Scaling: Multiply all px values in text-shadow by:
  • Mobile (320px): ×0.6
  • Tablet (768px): ×0.8
  • Desktop (1024px+): ×1.0
Accessibility Checklist
  • ✓ Ensure 4.5:1 contrast ratio (WCAG AA minimum)
  • ✓ Test with color blindness simulators (glitch/neon effects)
  • ✓ Keep text readable with heavy shadows
  • ✓ Use text-shadow for visual enhancement, not information
  • ✓ Test on real dark backgrounds, not just white text
