# All In One Helper (ytdlp-companion)

로컬 헬퍼 앱 — 영상 다운로드, 음성 인식, 배경 제거, NLE 내보내기 등

## 다운로드

[최신 릴리스 다운로드](https://github.com/groove1027/ytdlp-companion/releases/latest)

- **Windows**: `All In One Helper_x64-setup.exe`
- **macOS**: `All In One Helper_universal.dmg`

## 사용법

1. 위 링크에서 설치 파일 다운로드
2. 설치 후 실행 — 시스템 트레이에서 백그라운드 동작
3. [올인원 프로덕션](https://all-in-one-production.pages.dev) 웹사이트에서 자동 연결
